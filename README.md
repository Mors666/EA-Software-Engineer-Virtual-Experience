## 📌 Overview

This repository contains my submission for the **Electronic Arts (EA) Software Engineer Virtual Experience Program** offered by [Forage](https://www.theforage.com/). In this simulation, I took on the role of a **Junior Software Engineer** and completed a series of real-world development tasks involving game mechanics and feature implementation for *EA Sports: College Football* and *The Sims 4*.

---

## 🧠 Key Skills & Concepts

- 🧱 Object-Oriented Programming (OOP) in C++
- ✏️ Writing Feature Proposals
- 📊 Designing Class Diagrams (UML)
- 📁 Header File Structure & Class Definitions
- 🛠️ Code Optimization using `std::vector`
- 🐞 Debugging Live Game Features
- 💾 Memory Management in C++

---

## 🛠️ Tasks Completed

### 🔹 Task 1: Feature Proposal
> Proposed a **"Dynamic Coach Strategy Mode"** for EA Sports: College Football to increase strategic player control.

### 🔹 Task 2: Class Diagram
> Designed the system architecture with 10+ interrelated classes using UML.

### 🔹 Task 3: Header Files
> Implemented header files with class definitions and proper encapsulation.

### 🔹 Task 4: Bug Fix & Optimization
> ✅ Replaced legacy C-style arrays with `std::vector`  
> ✅ Fixed item removal bug in the inventory system  
> ✅ Improved memory management and overall readability
